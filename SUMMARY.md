## 5. КЛЮЧЕВЫЕ ПОЛНЕННЙМОСТИ

### Логические Операторы
Все три оператора теперь работают:
- [✅] **and** - логическое AND
- [✅] **or** - логическое OR
- [✅] **not** - логическое NOT

Пример:
```python
if a > 5 and b < 10:
    print("Both true")
if x or y:
    print("At least one")
if not condition:
    print("Negated")
```

### Атрибуты и Индексация

[✅] **obj.attr** - атрибуты объектов:
```python
class Point:
    def __init__(self):
        self.x = 10
p = Point()
print(p.x)  # НАПОНЕ работает!
```

[✅] **obj[index]** - индексация:
```python
arr = [1, 2, 3]
print(arr[0])  # НОВО работает!
```

[✅] **obj.method()** - вызовы методов:
```python
my_dict = {"a": 1}
for k in my_dict.keys():
    print(k)
```
